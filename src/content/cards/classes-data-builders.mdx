Python has several builders for simple records. The production question is not “which is newest?” but “what behavior do callers need?”

```python
# [CURRENT - 3.10-3.14] Works on Python 3.x
from dataclasses import dataclass
from typing import NamedTuple

class Point(NamedTuple):
    x: float
    y: float

@dataclass
class MutablePoint:
    x: float
    y: float

print(Point(1.0, 2.0))
print(MutablePoint(1.0, 2.0))
```
