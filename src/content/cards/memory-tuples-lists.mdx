Tuples are immutable records with exact allocation. Lists are mutable arrays of references with spare capacity for amortized appends.

```python
# [CURRENT - 3.10-3.14] Works on Python 3.x
t = (1, 2, 3)
same = tuple(t)   # same object

l = [1, 2, 3]
copy = list(l)    # new object

print(same is t)
print(copy is l)
```

<Callout type="tip">
  Use tuples for fixed records and return values. Use lists when the collection is meant to grow or change.
</Callout>
