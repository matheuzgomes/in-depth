Lists are general-purpose containers. For numeric data, queues, and streaming pipelines, specialized containers avoid hidden allocation costs.

```python
# [CURRENT - 3.10-3.14] Works on Python 3.x
from array import array
from collections import deque

readings = [21.5, 21.75, 22.0]
samples = array("d", readings)    # packed C doubles
window = deque(maxlen=60)         # O(1) append and popleft
```
