Dictionaries are fast because keys map through a hash table. The cost is memory overhead and a hard requirement: keys must be hashable.

```python
# [CURRENT - 3.10-3.14] Works on Python 3.x
cache = {}
cache[("user", 42)] = "active"  # tuple of hashable values

try:
    cache[["user", 42]] = "active"
except TypeError as exc:
    print(exc)
```
