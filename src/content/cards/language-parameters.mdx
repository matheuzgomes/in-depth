Function signatures are part of your API. Positional-only and keyword-only markers let you make invalid calls impossible.

```python
# Requires Python 3.8+ [PEP 570]
def connect(host, /, *, timeout=3.0, retries=2):
    return host, timeout, retries

connect("db.internal", timeout=1.5)  # clear

try:
    connect(host="db.internal")
except TypeError as exc:
    print(exc)
```
