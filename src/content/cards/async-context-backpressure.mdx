Real async production work is not just `await`. You also need backpressure and offloading patterns such as semaphores, `as_completed`, and thread delegation for blocking boundaries.

```python
# [OLDER / 3.7-3.8, CURRENT - 3.10-3.14] Works on Python 3.7+
import asyncio

async def fetch_one(name, sem):
    async with sem:
        await asyncio.sleep(0.1)
        return name
```

<Callout type="warn">
  Async code without concurrency limits is a reliability bug, not a sophistication badge. Bound fan-out explicitly.
</Callout>
